# RuleBot-AI

A simple, rule-based chatbot designed to answer questions on core Computer Science topics.

## ✨ Features

-   **Interactive Chat Interface**: Ask questions and get answers instantly.
-   **Supported Topics**:
    -   🤖 **AI** (Artificial Intelligence)
    -   🧠 **ML** (Machine Learning)
    -   🗄️ **DBMS** (Database Management Systems)
    -   🌳 **DSA** (Data Structures & Algorithms)
    -   🌐 **CN** (Computer Networks)
    -   🛠️ **SE** (Software Engineering)
    -   💻 **CL** (Computer Languages)

## 🚀 How to Run

1.  **Clone the repository** or download the `chat.html` file.
2.  **Open `chat.html`** in any modern web browser (Chrome, Firefox, Edge, etc.).
3.  **Start chatting!** Ask about general terms like "AI", "ML", "DSA", etc.

## 🛠️ Tech Stack

-   **HTML5**: Structure
-   **CSS3**: Styling (Responsive Design)
-   **JavaScript (Vanilla)**: Chatbot Logic & DOM Manipulation
